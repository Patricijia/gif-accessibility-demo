GIF Accessibility Extension

A Chrome extension that automatically generates alt-text descriptions for GIFs using a fine-tuned Vision-Language Model running entirely in the browser.

↓ Download Extension See Examples

How it works

1

Detect GIFs

The extension scans the page for animated GIFs as they load.

2

Extract frames

16 evenly-spaced frames are sampled and arranged into a 4×4 grid image.

3

Run the model

SmolVLM-256M (fine-tuned on GIF grids) runs locally via WebGPU — no data leaves your browser.

4

Apply alt text

The generated caption is written to the alt attribute, making the GIF accessible to screen readers.

Installation

  1. Download the extension using the button above and extract the .zip.
  2. Open Chrome and go to chrome://extensions.
  3. Enable Developer mode (top-right toggle).
  4. Click Load unpacked and select the extracted dist/ folder.
  5. Visit any page with GIFs — captions appear automatically.

Requires Chrome 120+. First load downloads the model (~500 MB, cached afterwards).

Launch Chrome with WebGPU enabled

If the extension falls back to WASM (slow), launch Chrome with these flags to force WebGPU:

google-chrome --enable-unsafe-webgpu --enable-features=Vulkan,VulkanFromANGLE,DefaultANGLEVulkan
chrome.exe --enable-unsafe-webgpu --enable-features=Vulkan
open -a "Google Chrome" --args --enable-unsafe-webgpu --enable-features=Vulkan
